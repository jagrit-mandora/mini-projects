# Alarm Clock

## Features
- User inputs alarm time (HH:MM:SS)
- Live clock display
- Input validation
- Plays an alarm sound when the time is reached

## Concepts Practiced
- Functions
- Loops
- User input
- `time` module
- `pygame`
- Terminal output (`\r`, `flush=True`)
- Input validation

## Future Improvements
- Multiple alarms
- Snooze feature
- Stop alarm with a key press
- GUI version
